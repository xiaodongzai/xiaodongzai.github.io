使用github搭建博客（1）

  • 前端开发

使用过两种博客引擎，jekyll及hexo。jekyll貌似比较火，但在window系统上环境比较麻烦，需要配置python,ruby等等，
而且很容易出错，当然熟悉ruby的可以玩玩。
jekyll比较好的教程：
阮一峰的搭建一个免费的，无限流量的Blog——github Pages和Jekyll入门
也可以直接使用 jekyllbootstrap
Jekyll中的语法高亮：Pygments

使用jekyll引擎搭建可以顺便学习一些github的常用操作，而hexo则比较傻瓜式，安装几步搞定，官方的教程基本够看，但还是稍微走了点弯路，有几点需要注意：

新建的github的Repository最好为： github用户名+github.io(如 username.github.io)，使后面配置更容易，而且不容易出错(主要是文件路径错误)
按照上述，发布后直接输入username.github.io即可访问
hexo server -g 命令启动服务器，期间更改页面内容可以直接看到效果
